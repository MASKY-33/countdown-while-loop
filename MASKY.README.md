# Countdown While‑Loop
This program uses a while‑loop to count down from 10 to 1 by decreasing the value each loop.

## Features
- Starts counting at 10
- Uses a while loop to decrease the number
- Stops when the value reaches 1
- Prints the final result


## Learning Purpose
A small exercise to practice:
- while‑loops
- decrementing values
- loop conditions
